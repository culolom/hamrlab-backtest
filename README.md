# 倉鼠回測平台

以 Streamlit 建立的多策略回測 UI。主畫面負責策略清單與說明，實際回測操作集中在各策略頁面，方便未來新增與擴充。

## 執行方式

```bash
pip install -r requirements.txt
streamlit run Home.py
```

## 內建策略
- **200SMA 回測基礎版**：輸入單一標的，透過 200 日（可調）SMA 產生進出場訊號，提供價格/均線圖、資金曲線與績效表。 
- **0050 LRS 槓桿策略**：原型 ETF（0050/006208）產生 200 日 SMA 訊號，實際交易正 2 槓桿 ETF，並與 Buy & Hold 做三組資金曲線比較。

主畫面在左側列出策略、右側顯示介紹並提供連結，回測功能請在各策略頁面操作。
